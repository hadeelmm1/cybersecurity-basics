# Nmap Basic Scan Report

## Objective
The objective of this project is to perform a basic network scan using Nmap to identify open ports and running services on a target system.

## Tool
- Nmap
- Kali Linux

## Command Used
nmap -sS -p 1-1000 scanme.nmap.org

## Steps
1. Open the terminal in Kali Linux.
2. Run the Nmap command.
3. Analyze the open ports and services.

## Result
The scan identified open ports that can be analyzed further for security assessment.
